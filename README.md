# Altaf Simavatwala — Portfolio & Resume Site

A multi-page personal portfolio and resume website for **Altaf Simavatwala**, focused on Risk Adjustment, healthcare operations, data analytics, and automation leadership.

## What It Does

Serves as an interactive resume and portfolio hub — showcasing professional background, impact metrics, case studies, and operator journey in a web format that goes far beyond a PDF.

## Key Features

- **Hero section** with rotating lens statements (leader / builder / analyst)
- **Impact metrics** — quantified outcomes from past roles
- **Why-hire reasons** clearly articulated for hiring managers
- **Experience timeline** with role-by-role breakdown
- **4 flagship case studies** with context, approach, and results
- **Skills page** — technical, operational, and domain expertise
- **Solutions page** — how Altaf approaches business problems
- **Operator journey** — narrative of growth as an operator
- **Projects page** — links to built tools and systems
- **Custom domain** via CNAME (asimavatwala.site)
- **Sync script** (`sync-site.sh`) for deploying updates

## Tech Stack

- HTML, CSS, Vanilla JavaScript
- GitHub Pages with custom CNAME domain
